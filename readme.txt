zeList importer is a generic plugin to import existing directories into zeList, WordPress Links Directory.
Right now (v0.1), a FreeGlobes importer is available.

== Installation ==

1. Upload plugin dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Import your old one.
